# EDUNUSA Presensi

Sistem manajemen kehadiran sekolah modern dan komprehensif berbasis React TypeScript.

## 🚀 Fitur Utama

### Dashboard
- Statistik real-time kehadiran siswa dan guru
- Grafik tren kehadiran mingguan/bulanan
- Activity feed untuk monitoring aktivitas terbaru
- Distribusi kehadiran dengan pie chart

### Absensi Real-time
- Monitoring kehadiran siswa secara real-time
- Tracking lokasi absensi dengan GPS
- Status kehadiran (Hadir/Terlambat/Tidak Hadir)
- Filter dan pencarian data

### Manajemen Data
- **Siswa**: CRUD data siswa lengkap dengan status
- **Guru**: Pengelolaan data guru dan mata pelajaran
- **Jadwal**: Pengaturan jadwal dan shift pembelajaran
- **Lokasi**: Setting lokasi valid untuk absensi

### Laporan & Analytics
- Export data ke Excel/PDF
- Grafik analisis kehadiran
- Laporan custom berdasarkan periode
- Statistik performa kehadiran

### Notifikasi & Alert
- Notifikasi otomatis untuk keterlambatan
- Alert untuk orang tua/wali
- Sistem notifikasi multi-channel (Email/SMS/WhatsApp)
- Log notifikasi terkirim

### Manajemen Izin
- Pengajuan izin online
- Verifikasi dan approval workflow
- Upload dokumen pendukung
- Tracking status izin

### Keamanan
- Audit trail lengkap
- Monitoring aktivitas user
- Backup otomatis
- Role-based access control

## 🛠️ Teknologi

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **Icons**: Lucide React
- **State Management**: React Hooks
- **Build Tool**: Create React App
